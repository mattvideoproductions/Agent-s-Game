# Bathtub Fluid with Ducks

A tiny browser demo using a simple stable-fluids style 2D simulation with floating cartoon ducks.

- Drag on the canvas to pour and stir water
- Shift+drag to drain
- Click "Add Duck" to spawn more
- Toggle Rain for a light drizzle
- Tweak Flow, Viscosity, Buoyancy

## Run

Open `index.html` in any modern browser. No build step needed.

If you prefer a quick static server:

```bash
# Python 3
cd /workspace && python3 -m http.server 8000
# Then open http://localhost:8000 in your browser
```
